Description:
It is a Hospital Managemnt Application using Java Swing.
There are 4 roles- System Admin, Community Admin, Person and Doctor.
System Admin has access to all the data.
Community Admin only can view the Community data i.e. number of persons in the community and the number of patients.
Person can add his data and register himself. He can edit, view and delete data.
Doctor can add patients, add the vital signs and increase the encounter counts.
